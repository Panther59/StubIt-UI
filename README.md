# StubIt UI

Angular management front-end for StubIt.

## Folders
- /src: Angular SPA source code
- /e2e: End-to-end UI tests
- /specs: Executable UI specs (SpecKit + Node.js)
- /docs: UI docs and requirements

## Setup

### Angular (Development)
```
cd src
npm install
ng serve
```
### E2E Tests
```
cd e2e
npm install
npm run e2e
```
### Specs
```
cd specs
npm install
npx speckit run
```
